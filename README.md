A simple and fully functional CRUD (Create, Read, Update, Delete) web application built using the Laravel framework.
This project demonstrates backend development using MVC architecture, Eloquent ORM, migrations, and MySQL database integration.

📌 Features

➕ Create New Records

📄 Read / View Records

✏️ Update Existing Records

❌ Delete Records

✅ Server-side Form Validation

🗄️ Database Migrations

🧩 MVC Architecture Implementation

🎨 Clean UI using Blade Templates

🛠️ Tech Stack

PHP

Laravel

MySQL

HTML5

CSS3

Blade Templating Engine
